# KanUUNTt sound pack

Put your own MP3 files in this folder with these exact filenames:

- `question-start.mp3` - played when a question opens.
- `countdown-loop.mp3` - looped while the normal countdown runs.
- `urgent-countdown-loop.mp3` - looped during the last 5 seconds.
- `time-up.mp3` - played when the countdown reaches 0.
- `answer-reveal.mp3` - played when the correct answer is shown.
- `scoreboard.mp3` - played when the scoreboard is shown.

Use audio files you own or have permission to use. If a file is missing or cannot be played, the display falls back to the built-in synthesized quiz sounds.
